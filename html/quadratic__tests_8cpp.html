test_state_t test_solving_quadratic ( int * tests_number,
int * errors_number,
const char * filename )

  • Run tests of solve_quadratic(...)

  • Expected values are taken from file "tests.txt" in form:
    'a b c x1 x2 roots_number'.
    • Meaning of values:
      • a, b and c are coefficients of equation ax^2 + bx + c == 0.
      • x1 and x2 are roots of these equation.
      • roots_number is number of roots.
    • Don't skip roots if number is less then 2.
    • Function does not compare second root if roots_number == 1.
    • Function does not compare roots if there are zero or infinitely many roots.
    • If equation has infinitely many roots type in -2
    • Function returns:
      • NO_SUCH_FILE if there is no file "tests.txt".
      • INVALID_LINES if there is error in "tests.txt".
      • SUCCESS_TEST if all test provided in "tests.txt" were carried out.
        Parameters
        [out]tests_numberPointer to ineteger in which function will put total number of tests.
        [out]errors_numberPointer to integer in which function will put total number of errors.
        Returns
        Error (or success) code.
