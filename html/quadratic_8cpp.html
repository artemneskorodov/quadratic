◆ solve_linear()

solving_state_t solve_linear ( quadratic_equation_t * equation)
static

  • Function solves linear equation bx + c == 0, where b and c are fields of equation struct

  • Function returns:
    • SOLVING_SUCCESS (if solved equation successfully)
    • SOLVING_ERROR (in case of unexpected error)
    • INVALID_COEFFICIENTS (if one of coefficients is not finite number)
    • There are no other return values

Function write root to fields 'x1' and 'x2' of equation struct.

  • Function writes number of roots in field 'number':
    • NOT_SOLVED can occure there only before structure was sent to solve_quadratic() or in case of error
    • NO_ROOTS if equation has no real roots
    • ONE_ROOT if equation has one real root
    • INF_ROOTS if equation has infinitely many roots
    • TWO_ROOTS can't occure in case of linear equation

◆ solve_quadratic()

solving_state_t solve_quadratic ( quadratic_equation_t * equation)

  • Solves quadratic equation in form ax^2 + bx + c == 0

  • Function gets coefficients from fields a, b and c of equation struct.
    • It solves linear equation if a == 0.
    • Function returns:
      • SOLVING_SUCCESS (if solved equation successfully)
      • SOLVING_ERROR (in case of unexpected error)
      • INVALID_COEFFICIENTS (if one of coefficients is not finite number)
      • There are no other return values
    • Function writes roots in fields 'x1' and 'x2' of equation structure.
    • Number of roots is written in field 'number':
      • NOT_SOLVED can occure there only before structure was sent to solve_quadratic() or in case of error
      • NO_ROOTS if equation has no real roots
      • ONE_ROOT if equation has one real root
      • TWO_ROOTS if equation has two real roots
      • INF_ROOTS if equation has infinitely many roots
Parameters
[out]equationPoint to equation structure, containing coefficients of quadratic equation
