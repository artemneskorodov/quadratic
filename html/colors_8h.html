  • Allows to print colored text.

  • Use only supported colors: RED, GREEN, YELLOW, BLUE, PURPLE, CYAN, WHITE, DEFAULT
    • String can be pointer to constant string, use of formats and arguements are same as in printf():
      • 'c' – character symbol.
      • 'd' – decimal integer.
      • 'i' – decimal integer.
      • 'e' – floating point number in exponential form.
      • 'f' – floating point number in decimal form (length of tail can be controlled by '.num' between '' and 'f').
      • 'g' – floating point that chooses the smallest way to write between 'e' and 'f'.
      • 'o' – octal integer.
      • 's' – string.
      • 'x' – hexadecimal integer.
      • 'p' – pointer.
      • 'n' – number of read symbols.
      • 'u' – unsigned integer.
      • '%[]' – scans many symbols.
      • 'l...' – long integer or double.
