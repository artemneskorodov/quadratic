◆ read_line()

reading_state_t read_line ( FILE * file,
quadratic_equation_t * equation )

  • Reads line of tests file.

  • Format of reading: "%lg %lg %lg %lg %lg %d".
    • Inputs are considered as "a b c x1 x2 n_roots", where.
      • a, b and c are coefficients of quadratic equation ax^2 + bx + c == 0.
      • x1 and x2 are roots of these equation.
      • n_roots is expected number of roots.
    • Note that roots are not compared if n_roots == -2 || n_roots == 0\ and only x1 is compared if n_roots == 1.
    • Read documentation of test_solve_quadratic().
    • Puts result of reading in equation structure.
    • Function returns:
      • READING_SUCCESS if it read coefficients successfully.
      • READING_ERROR if it read coefficients unsiccessfully.
      • READING_END if pointer in file reached the EOF.
        Parameters
        [in]filePointer to opened file where tests are located.
        [out]equationPointer a structure where function puts coefficients, expected roots and roots number.
        Returns
        Error(or success) code
